# Bài Test: Web 

Dự án sử dụng **React + TypeScript + Vite** để tạo giao diện **Web 360 độ**, với thư viện [@paulin_bixlers/react-360-product-viewer](https://www.npmjs.com/package/@paulin_bixlers/react-360-product-viewer) để hiển thị hình 360.

## Công nghệ sử dụng
- React + TypeScript
- Vite
- TailwindCSS
- @paulin_bixlers/react-360-product-viewer
- ESLint cơ bản cho React + TypeScript


## Cách chạy
```bash
npm install
npm run dev
```
## Tính năng

- Xem ảnh 360 độ, xoay bằng chuột hoặc cảm ứng

- Overlay hướng dẫn khi lần đầu mở trang

- Menu dưới cùng có thể mở/đóng

- Quản lý trạng thái overlay và menu bằng React state
